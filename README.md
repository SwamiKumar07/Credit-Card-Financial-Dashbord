# Credit-Card-Financial-Dashbord
Power Bi Dashbord

Developed an interactive Credit Card Financial Dashboard using Power BI to analyze customer spending and transaction behavior.

Cleaned, transformed, and modeled raw data using SQL and Excel to ensure accurate reporting.

Analyzed key metrics such as credit limit, utilization ratio, transaction amount, and delinquent accounts.

Created dynamic visualizations (charts, KPIs, slicers) to track weekly and quarterly performance trends.

Improved business insights by identifying high-risk customers and profitable spending categories.
